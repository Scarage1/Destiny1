# 11 - PR and Issue Template Content

## PR template body
- Summary
- Type of change
- Acceptance criteria mapping
- Validation evidence
- Risk and rollback
- AI-assisted development log
- Final checklist

## Feature issue format
- Goal
- Non-goals
- Scope (max 5 files)
- Acceptance criteria
- Validation steps
- Risks
- Dependencies

## Bug issue format
- Problem
- Reproduction steps
- Expected vs actual
- Impact
- Fix scope
- Validation
- Risk

## Recommended placement in repository
- .github/pull_request_template.md
- .github/ISSUE_TEMPLATE/feature.md
- .github/ISSUE_TEMPLATE/bug.md
